CRM & Sales Pipeline System

This project is a lightweight Customer Relationship Management (CRM) system originally designed and implemented in Google Sheets. It models customers, sales opportunities, and activity tracking using structured data relationships and light automation.

Key Features:

  • Customer, opportunity, and activity data model with unique IDs
  • Stage-based sales pipeline with automated probabilities
  • Forecasted revenue calculations
  • Overdue follow-up tracking for activities
  • Executive sales dashboard with KPIs and funnel analysis

The system was exported as a read-only Excel version for GitHub hosting, while the live Google Sheets version supports full interaction and automation.
