Data Cleaning & Preparation

Cleaned and prepared a real-world dataset containing inconsistent formatting, missing values, and duplicates to make it analysis-ready for reporting and dashboards.

Key Features:

  • String standardization (Country, Gender, Residence, Education Level)
  • Null value handling (filled missing scores)
  • Type correction for numeric and text columns
  • Duplicate removal and data validation
  • Verification using filters and conditional formatting

Outcome:

Produced a clean, structured dataset ready for analysis and visualization—demonstrating Excel + Power Query skills used in data analyst and reporting roles.
